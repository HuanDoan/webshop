# OpenCart 4 & Docker

## Overview

[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.0-8892BF.svg?style=flat-square)](https://php.net/)
[![GitHub release](https://img.shields.io/github/v/release/opencart/opencart)](https://github.com/opencart/opencart)

OpenCart is a free open source ecommerce platform for online merchants. OpenCart provides a professional and reliable foundation from which to build a successful online store.

This repository is running the opencart 4 within docker by docker-compose.


## How to install

```shell
docker-compose up --build -d
```

The local server is running in `localhost:8080`.

## Getting started
The admin dashboard is located at
```
http://localhost/admin-dashboard
```

The default admin user:
```
admin
12345678
```

The database's root user & user, please check it in `docker-compose.yml`
